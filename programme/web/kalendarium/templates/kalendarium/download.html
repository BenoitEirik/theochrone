Aide à l'export

Le module d'export vous permet de récupérer une année entière avec ses fêtes pour pouvoir être utilisé dans des programmes comme Outlook, Google Calendar, Thunderbid, etc.
Nous vous suggérons fortement l'utilisation du format ICS, qui sera reconnu partout.
Pour importer le fichier exporté dans votre calendrier, consultez sa documentation. Une simple recherche sur Google devrait vous donner toutes les clefs.

Si vous voulez exporter le Théochrone au format PDF, ou Excel, ou si vous désirez exporter une période autre qu'une année, téléchargez le programme sur votre ordinateur.
